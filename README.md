# Evidence pojištění – Frontend

React frontend pro webovou aplikaci evidence pojištěnců a jejich pojištění.

## Tech Stack

| Vrstva | Technologie |
|--------|-------------|
| Frontend | React 18 + Vite |
| HTTP klient | Fetch API |
| Styling | CSS inline |

## Funkce

- Zobrazení seznamu pojištěnců
- Přidání / editace / smazání pojištěnce
- Zobrazení pojištění daného pojištěnce
- Přidání / editace / smazání pojištění

## Spuštění

1. Nainstaluj závislosti:
```bash
npm install
```
2. Spusť frontend:
```bash
npm run dev
```
3. Frontend běží na `http://localhost:5173`
